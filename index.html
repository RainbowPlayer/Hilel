GitHub

  1. git status -Перераховує всі нові або змінені файли, які потребують фіксації
  2. git add . -Індексує вказаний файл для наступного коміту
  3. git commit -m "" -Фіксує проіндексовані зміни та зберігає їх в історію версій
  4. git push origin (your branch) -Завантажує всі зміни локальної гілки у віддалений репозиторій
  5. git checkout -b (your branch) -Переводить на іншу гілку
